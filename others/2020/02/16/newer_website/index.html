ホームページの大規模更新

Posted by : Keisuke Yanagisawa at Feb 16, 2020

Category : Others

前回ホームページを大幅に更新したのは2017年ごろなのだが、顔写真が無かったり、ちょっと殺風景だったり、多言語対応が無理やりだったりしたのでそのあたりも対応できるようにした。

それと併せて、これまではblogがちゃんと機能していなかったのでhidden状態になっていたのだけれど、これも公開することに。あ、残念ながら日本語のみ対応の予定です。

論文を読んだメモとか、これまではてブロにまとめてたけど、これからはこちらにまとめていけたらいいなぁと思いつつ、でもどのくらいまじめに更新するかなーあ。

あ、Qiitaなどはこれからも必要に応じて使う予定です。適材適所で、使い分けていきたい。
